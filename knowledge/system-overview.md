# AITuber Robot Controller

## 概要

完全ローカルで動くAIアシスタント × ロボットMVP
ロボットアームで一緒にETごっこもできるよ！（うまくいかないこともあるかもしれないけど）

## できること

- AIキャラクターと音声で会話できる
- 「ETごっこしよう」と話しかけると、ロボットアームが指先にタッチしてくれる
- このシステム自体についても質問できる（RAGで回答）

## デモの流れ

1. 「ETごっこしよう」と話しかける
2. AIキャラクターが「指を出してね！」と返答する
3. 人差し指を出す
4. ロボットアームがカメラで指を認識して、指先にタッチする

※「ETごっこしよう」という普段使わない言葉をトリガーにすることで、誤検知を防いでいる
